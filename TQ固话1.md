### 二. 智能固话使用与功能介绍（助理小电）</br></br>	
#### 1. 登陆  
&emsp;&emsp;点击助理小电APP图标，会提示登录页面，根据TQ公司提供的个人账号和密码进行登录即可。</br>

##### 2.1 首次登陆</br> 
&emsp;&emsp;助理小电首次登录后，手机系统会提示请开启辅助功能，根据下图步骤开启即可</br>
![][id8]</br>
##### 2.2 使用2G/3G/4G网络同步</br> 
&emsp;&emsp;登录后，点开设置页面，可设置录音文件同步选项：“使用2G/3G/4G网络同步”；</br>   
&emsp;&emsp;默认我们是wifi网络下自动同步上传。  
![][id9]</br>
##### 2.3 呼入/呼出结束自动弹屏</br> 
&emsp;&emsp;助理小电设置呼入/呼出结束自动弹屏按钮显示在我页面-------设置。设置如下图：</br> 
![][id10]</br>
##### 2.4 固话端弹屏效果</br> 
&emsp;&emsp;点开后，接打电话挂断后会弹出新建/编辑页面；新建页面是为陌生号码呼入或者呼出，编辑页面是为已存客户信息呼入或者呼出。</br>
&emsp;&emsp;通过固话端弹屏页面进行编辑跟进备注等信息保存之后，该客户信息均能同步到TQ的CRM后台（前提是网络通畅的情况下），如下图：</br> 
![][id11]</br>
##### 2.5 拨号前缀</br> 
&emsp;&emsp;助理小电为满足部分客户，新增拨号加前缀功能，拨号前缀功能使用显示在我页面--------设置；默认为空值，需根据客户需求自己设置。如下图：</br> 
![][id12]</br>

#### 3. 号及拨号方式（拨号页面）  
##### 3.1 PC端点击外呼拨号</br> 
&emsp;&emsp;助理小电为满足广大客户需求，研发点击外呼功能；通过智能固话登录对应的TQ号，在后台保存此工作手机对应的UUID（固话唯一标识），当客户在后台或者客户端点击外呼的时，会把外呼的消息推送到对应的智能固话上，然后调用智能固话的外呼方法，智能固话呼出电话外呼成功。点击外呼功能需要连接网络（4G或者WIFI网络，视情况而定）。</br>
&emsp;&emsp;后台登录地址: [http://vip.mobile.tq.cn/vip/](http://vip.mobile.tq.cn/vip/ "登陆地址") </br>
![][id15]</br>
##### 3.1.1 外呼弹屏</br> 
&emsp;&emsp;当座席在后台或者客户端点击外呼的时候，会出现弹屏页面，已有客户会弹出该客户的客户详细信息页面，座席可根据沟通过程对该客户信息进行进一步的填写跟进备注信息。未知客户会弹出新建页面，可在弹出的新建页面进行编辑保存客户信息，业务快速处理客户信息，如下图：</br> 

![][id16]</br>
##### 3.2 用数字键</br> 
&emsp;&emsp;输入电话号码，然后按拨号键开始呼叫。如果需要更改电话号码，按删除键删除输入的字符。呼叫时显示屏会显示拨打电话。对方摘机后显示屏上会显示通话状态信息。</br>
&emsp;&emsp;通话结束后请按屏幕“挂号”图标挂断电话。</br>
##### 3.3 使用触摸屏按键</br> 
&emsp;&emsp;选择“电话”图标，点击右上角，弹出拨号界面，输入号码，点击拨号图标进行呼叫。对方摘机后显示屏幕上会显示通话状态信息。</br>
&emsp;&emsp;通话结束后请按屏幕“挂号”图标挂断电话。</br>
##### 3.4 使用耳机键</br> 
&emsp;&emsp;按耳机键，然后通过数字键或者使用触摸按键经行拨号，输入电话号码，然后按拨号键或点击拨号图标开始呼叫。对方摘机后显示屏上会显示通话状态信息。</br>
##### 3.5 接听电话</br> 
&emsp;&emsp;正常情况下，收到来电时，按拨号键或”接听”键都可以接听；也可以直接滑动屏幕电话按钮，进行接听。</br>
##### 3.6 拒接电话</br>
&emsp;&emsp;有来电时，按挂号键可以拒接电话；或者互滑动屏幕电话按钮，选择挂断或回复短信。</br>

###四. 注意事项</br></br>	
&emsp;1. 固话进行外呼时，请不要点击固话自带的录音功能，会和我们的软件相冲突，出现卡死状态。</br>
&emsp;2. 请使用TQ号和密码登录“助理小电”，且在通话结束后保持固话网络通畅。“助理小电”是自动双向通话录音软件，拨打电话即可录音，无需做其他操作。在通话结束后，通过固话流量网络或者WIFI网络自动上传电话记录及录音。</br>
&emsp;3. 请先卸载固话中与“助理小电”有冲突的录音软件，然后再使用“助理小电”（注：固话中安装的多个录音软件，相互之间会有影响，会导致录音助手无法录音）。</br>
&emsp;4. 如果能够正常录音，请不要随意的修改参数设置（可能会导致固话无法录音）。使用智能固话，请不要下载其他安全软件，如：360手机卫士等。</br>
&emsp;5. 智能使用过程中出现问题后，需要客户配合查看固话系统版本，那么查看系统版本的方法是在设置里查看。操作过程为“设置--->关于手机（或者我的MIUI）--->然后截图或者拍照”。</br>
&emsp;6. 工作手机使用过程中出现问题后，需要客户配合查看小电的版本号，查看方式： “设置--->应用）--->>>找到助理小电--->然后截图或者拍照”</br>

###五. 常见故障与原因</br>


|    故障现象    |                   可能原因                   |
|:----------------:|:----------------------------------------------:|
| 不能开机       | 1.电池电量不足                               |
|                | 2.适配器未连接                               |
| PIN码出错      | 1.插入错误的USIM卡                           |
|                | 2.USIM卡已被加锁，必须由专业人员初始化电话机 |
| 通话过程“掉线” | 1.电池电量不足                               |
|                | 2.信号强度不够                               |
| 通话中调节音量 | 按音量+/-键进行调节                          |
| 设备上不了网   | 1.WLAN未连接                                 |
|                | 2. USIM卡欠费，数据开关是否打开              |
| LAN口上不了网  | 1.确保USIM卡不欠                             |
|                | 2.确保设备的流量开关为打开状                 |
|                | 3.电脑上的IP地址获取方式为“自动获取          |
|                | 4.电源适配器未连接                           |
